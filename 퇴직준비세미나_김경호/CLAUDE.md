# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

이 저장소는 **퇴직준비세미나** 10회차 교육과정의 학습용 Obsidian Vault입니다. 소프트웨어 프로젝트가 아닌 마크다운 기반 지식 관리 시스템으로, AI 도구 활용과 개인 지식 관리(PKM)를 학습하는 것이 목적입니다.

- **원격 저장소**: https://github.com/tonuri/project.git

## Repository Structure

PARA 방법론(Projects/Areas/Resources/Archive)에 따라 폴더가 구성되어 있습니다.

- `Projects/` — 마감 기한이 있는 활성 프로젝트
- `Areas/` — 지속적으로 관리하는 관심 영역
- `Resources/` — 참고 자료 및 도구 사용법
- `Archive/` — 완료되었거나 비활성화된 항목
- `회차별 자료/` — 10회차 강의 자료 (1회차~10회차)
- `.obsidian/` — Obsidian 설정 파일 (직접 수정하지 말 것)

## Conventions

- 모든 콘텐츠는 **한국어**로 작성하되, 기술 용어는 영문 병기 가능
- 마크다운 파일은 Obsidian 호환 형식 사용 (위키링크 `[[노트명]]`, 태그 `#태그`)
- 파일명에 회차 번호를 접두사로 사용: `N회차_제목.md`
- Resources 폴더의 파일명 형식: `도구명 사용법.md`

## Key Curriculum Phases

1. **1~2회차**: 환경설정, 프롬프트 엔지니어링 기초
2. **3~4회차**: AI Agent 개념, 24시간 자동화
3. **5~6회차**: Obsidian 지식관리, Claude Skill 문서화
4. **7~8회차**: 1인 연구자 도구, 데이터 분석
5. **9~10회차**: 프로젝트 관리, T자형 역량 확장
